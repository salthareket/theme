Image Uploader v2.0 — Vanilla JS

Drag & Drop image uploader. No jQuery required. Same API, modern code.

Example 1 — Basic Usage


new ImageUploader('#uploader-1');

Example 2 — With Options & Preloaded


new ImageUploader('#uploader-2', {
    preloaded: [
        { id: 1, src: 'https://picsum.photos/500/500?random=1' },
        { id: 2, src: 'https://picsum.photos/500/500?random=2' },
        { id: 3, src: 'https://picsum.photos/500/500?random=3' },
        { id: 4, src: 'https://picsum.photos/500/500?random=4' },
    ],
    imagesInputName: 'photos',
    preloadedInputName: 'old',
    maxSize: 2 * 1024 * 1024,
    maxFiles: 10,
    onError: (msg) => console.warn(msg),
    onChange: (files, preloaded) => {
        console.log('Files:', files.length, 'Preloaded:', preloaded.length);
    }
});

Example 3 — jQuery Compat (if jQuery loaded)


// Works if jQuery is loaded

This theme requires the Timber plugin to function properly. Timber provides powerful tools for developing WordPress themes using the Twig templating engine.

To install Timber, please follow these steps:

  1. Ensure that Composer is installed on your server. You can download it from getcomposer.org.
  2. Navigate to your theme's directory using your terminal or command prompt.
  3. Run the following command:
composer require timber/timber

Once the installation is complete, refresh this page to continue using the theme.
